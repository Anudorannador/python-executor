# Summary

Generate concise, leader-friendly status summaries.

## Related (Soft)

Often used with:

- `learn` - when extracting a reusable summary skill from history
- `manifest` - when producing evidence via file-first artifacts

## Output

Use the template in `references/leader-summary-template.md`.

## Rules

- Keep it scannable: short sections + bullet points
- Lead with outcomes and impact
- Call out risks clearly with owners and mitigations
- Include links to evidence (PRs, dashboards, logs)
- If images are included, prefer small screenshots with captions

## Images

See `references/markdown-images.md` for safe, portable markdown image linking.
